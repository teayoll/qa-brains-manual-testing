QA BRAINS Practice Site — проект, направленный на практику ручного тестирования

## Что представляет из себя проект?

Проект представляет из себя репозиторий, содержащий документацию по итогам ручного тестирования веб-сайта QA BRAINS Practice Site.
Данный проект был выполнен в рамках подготовки портфолио для позиции Junior QA Engineer. Он призван продемонстрировать отработанные навыки тестирования и работы с сопутствующей документацией по тестированию.

## В чем цель проекта?

Цель проекта — получить практический опыт ручного тестирования реального веб-приложения, а также продемонстрировать умение:

* анализировать требования;
* планировать тестирование;
* создавать тестовую документацию;
* находить и описывать дефекты.

## Что является объектом тестирования?

* Объект: QA BRAINS Practice Site
* Тип объекта: веб-приложение
* Назначение объекта: учебная платформа для практики QA-тестирования

## Область тестирования

В рамках данного проекта выполняется тестирование:

* элементов пользовательского интерфейса;
* валидации вводимых данных;
* навигации между страницами;
* базовых пользовательских сценариев.

## Какие будут использованы типы тестирования?

* функциональное тестирование;
* UI-тестирование;
* smoke-тестирование;
* исследовательское тестирование.

## Какие будут использоваться инструменты?

* Google Sheets (чек-листы и тест-кейсы);
* Jira (баг-репорты);
* Chrome DevTools;
* GitHub.

## Описание структуры репозитория

* docs/            — проектная документация  
* test-artifacts/  — чек-листы, тест-кейсы, баг-репорты  
* screenshots/     — скриншоты дефектов
