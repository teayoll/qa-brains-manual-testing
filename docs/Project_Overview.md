## Обзор проекта

## Название проекта

QA BRAINS Practice Site — проект по ручному тестированию

## Объект тестирования

Веб-приложение: QA BRAINS Practice Site

## Описание приложения

QA BRAINS Practice Site — это общедоступное демонстрационное веб-приложение, созданное для практики навыков тестирования у QA-инженеров.
Приложение содержит несколько функциональных модулей:

* формы регистрации и авторизации пользователя;
* создание, редактирование и удаление задач;
* элементы пользовательского интерфейса (кнопки, чекбоксы, поля ввода);
* фильтры и сортировку данных.

Сайт специально включает потенциальные дефекты и пограничные случаи для учебного тестирования.

## Цели проекта

Основные цели проекта:

* получение практического опыта ручного функционального тестирования;
* отработка навыков создания QA-документации;
* практика поиска, воспроизведения и оформления баг-репортов;
* подготовка полноценного QA-проекта для портфолио Junior QA Engineer.

## Область тестирования

В проекте тестируются:

* элементы интерфейса и верстка страниц;
* валидация форм (логин, регистрация, поля ввода);
* навигация между страницами;
* основные пользовательские сценарии;
* сообщения об ошибках и проверки данных.

Вне области тестирования:

* нагрузочное тестирование;
* тестирование производительности;
* тестирование безопасности.

## Типы тестирования

* функциональное тестирование;
* UI-тестирование;
* smoke-тестирование;
* регрессионное тестирование;
* исследовательское тестирование.

## Тестовое окружение

* Операционная система: Windows 11 Home
* Браузер: Google Chrome (последняя версия)
* Разрешение экрана: Desktop

## Инструменты

* Google Sheets (чек-листы и тест-кейсы)
* Jira (баг-репорты)
* Chrome DevTools
* GitHub

## Ожидаемый результат

В результате выполнения проекта будет подготовлен полный комплект QA-документации, включая:

* тест-план;
* чек-листы;
* тест-кейсы;
* баг-репорты;
* итоговый отчет о тестировании.

Проект демонстрирует практические навыки, необходимые для позиции Junior QA Engineer.
